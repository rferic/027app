# 027Apps Docs

Documentazione della piattaforma per sviluppatori che costruiscono con 027Apps o lo integrano nei loro progetti.

027Apps è una piattaforma open-source per applicazioni di gruppo — per famiglie, team o qualsiasi collettivo. Ogni gruppo gestisce i propri utenti, ruoli e permessi delle applicazioni.

## Per iniziare

La piattaforma espone una REST API per l'autenticazione, la gestione degli utenti, l'accesso alle applicazioni e la configurazione delle lingue. Tutti gli endpoint sono versionati sotto `/api/v1/`.

Inizia con [Autenticazione](/doc/api/authentication) per ottenere un token di sessione, poi esplora gli endpoint disponibili.
