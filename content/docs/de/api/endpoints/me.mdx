# GET /api/v1/me

Gibt das Profil und die Gruppenmitgliedschaft des aktuell authentifizierten Benutzers zurück.

## Authentifizierung

**Erforderlich:** JWT (`Authorization: Bearer <token>`)

API-Schlüssel werden auf diesem Endpoint nicht akzeptiert. Die Antwort ist immer auf den durch den Token identifizierten Benutzer beschränkt.

## Anfrage

```
GET /api/v1/me
Authorization: Bearer <access_token>
```

Keine Abfrageparameter oder Anfragekörper.

## Antwort

**200 OK**

```typescript
type MeResponse = {
  id: string           // Supabase Auth user ID (UUID)
  email: string        // User's email address
  display_name: string | null
  avatar_url: string   | null
  locale: string       // User's preferred locale (e.g. "en", "es")
  role: 'admin' | 'member'
  joined_at: string    // ISO 8601 timestamp
  created_at: string   // ISO 8601 timestamp
  updated_at: string   // ISO 8601 timestamp
}
```

Beispiel:

```json
{
  "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "email": "alice@example.com",
  "display_name": "Alice",
  "avatar_url": null,
  "locale": "en",
  "role": "member",
  "joined_at": "2025-03-15T10:22:00.000Z",
  "created_at": "2025-03-15T10:22:00.000Z",
  "updated_at": "2025-04-01T08:00:00.000Z"
}
```

## Fehler

| HTTP | `error` | Ursache |
|---|---|---|
| 401 | `unauthorized` | JWT fehlt, ist abgelaufen oder fehlerhaft |
| 403 | `forbidden` | Token ist gültig, aber der Benutzer ist kein Mitglied einer Gruppe |

## Beispiele

### curl

```bash
curl https://example.com/api/v1/me \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

### fetch (TypeScript)

```typescript
type MeResponse = {
  id: string
  email: string
  display_name: string | null
  avatar_url: string | null
  locale: string
  role: 'admin' | 'member'
  joined_at: string
  created_at: string
  updated_at: string
}

const res = await fetch('/api/v1/me', {
  headers: {
    Authorization: `Bearer ${session.access_token}`,
  },
})

if (!res.ok) {
  const err = await res.json()
  throw new Error(err.message)
}

const user: MeResponse = await res.json()
```
