# 027Apps Docs

Platform documentation for developers building with or integrating 027Apps.

027Apps is an open-source platform for group apps — for families, teams, or any collective. Each group manages its own users, roles, and app permissions.

## Getting started

The platform exposes a REST API for authentication, user management, app access, and locale configuration. All endpoints are versioned under `/api/v1/`.

Start with [Authentication](/doc/api/authentication) to obtain a session token, then explore the available endpoints.
