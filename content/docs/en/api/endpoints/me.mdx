# GET /api/v1/me

Returns the profile and group membership of the currently authenticated user.

## Authentication

**Required:** JWT (`Authorization: Bearer <token>`)

API keys are not accepted on this endpoint. The response is always scoped to the user identified by the token.

## Request

```
GET /api/v1/me
Authorization: Bearer <access_token>
```

No query parameters or request body.

## Response

**200 OK**

```typescript
type MeResponse = {
  id: string           // Supabase Auth user ID (UUID)
  email: string        // User's email address
  display_name: string | null
  avatar_url: string   | null
  locale: string       // User's preferred locale (e.g. "en", "es")
  role: 'admin' | 'member'
  joined_at: string    // ISO 8601 timestamp
  created_at: string   // ISO 8601 timestamp
  updated_at: string   // ISO 8601 timestamp
}
```

Example:

```json
{
  "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "email": "alice@example.com",
  "display_name": "Alice",
  "avatar_url": null,
  "locale": "en",
  "role": "member",
  "joined_at": "2025-03-15T10:22:00.000Z",
  "created_at": "2025-03-15T10:22:00.000Z",
  "updated_at": "2025-04-01T08:00:00.000Z"
}
```

## Errors

| HTTP | `error` | Cause |
|---|---|---|
| 401 | `unauthorized` | Missing, expired, or malformed JWT |
| 403 | `forbidden` | Token is valid but the user is not a member of any group |


### fetch (TypeScript)

```typescript
type MeResponse = {
  id: string
  email: string
  display_name: string | null
  avatar_url: string | null
  locale: string
  role: 'admin' | 'member'
  joined_at: string
  created_at: string
  updated_at: string
}

const res = await fetch('/api/v1/me', {
  headers: {
    Authorization: `Bearer ${session.access_token}`,
  },
})

if (!res.ok) {
  const err = await res.json()
  throw new Error(err.message)
}

const user: MeResponse = await res.json()
```
