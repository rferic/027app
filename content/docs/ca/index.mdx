# 027Apps Docs

Documentació de la plataforma per a desenvolupadors que construeixen amb 027Apps o l'integren en els seus projectes.

027Apps és una plataforma de codi obert per a aplicacions de grup — per a famílies, equips o qualsevol col·lectiu. Cada grup gestiona els seus propis usuaris, rols i permisos d'aplicació.

## Primers passos

La plataforma exposa una API REST per a l'autenticació, la gestió d'usuaris, l'accés a les aplicacions i la configuració d'idiomes. Tots els endpoints estan versionats sota `/api/v1/`.

Comença per [Autenticació](/doc/api/authentication) per obtenir un token de sessió i explora els endpoints disponibles.
