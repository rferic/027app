# Instrucciones para Claude Code

Lee el archivo `AGENTS.md` en la raíz del proyecto. Contiene todas las convenciones del proyecto, incluyendo la gestión de sprints.

## Sprints

Los planes de sprint se guardan en `.plans/` con el formato `sprint-{N}-{descripcion}.md`.

- **Pendiente:** `sprint-{N}-{descripcion}.md`
- **Implementado:** `sprint-{N}-{descripcion}.md.done`

Antes de empezar cualquier tarea revisa `.opencode/plans/` para saber en qué sprint estás y continúa la numeración de tareas desde la última TASK del sprint más reciente.
